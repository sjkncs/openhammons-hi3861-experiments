# Lab3 PC 工具说明

## TCP&UDP Debug 工具

运行目录下的 `TCPUDPDbg.exe` 即可使用。

### 主要功能

1. **UDP 通讯**: 创建 UDP 套接字，发送/接收 UDP 数据包
2. **TCP 服务器**: 监听端口，接受 TCP 客户端连接
3. **TCP 客户端**: 连接远程 TCP 服务器
4. **多连接**: 支持同时管理多个 TCP 连接
5. **广播**: 支持 UDP 广播发送和接收
6. **HEX/ASCII**: 支持 HEX 和 ASCII 两种数据显示格式
7. **文件发送**: 支持发送文件和保存接收数据到文件

### 使用方法

#### UDP 模式

1. 选择 "UDP" 模式
2. 输入本地监听端口（如 50001）
3. 输入目标 IP 和端口
4. 点击 "打开"
5. 发送数据: `{"cmd":"forward"}`

#### TCP 服务器模式

1. 选择 "TCP 服务器" 模式
2. 输入监听端口（如 8080）
3. 点击 "打开"
4. 等待 Hi3861 连接
5. 发送数据测试

#### TCP 客户端模式

1. 选择 "TCP 客户端" 模式
2. 输入服务器 IP 和端口
3. 点击 "连接"
4. 观察数据收发

### 快捷键

- `Ctrl+L` 清空接收区
- `Ctrl+Enter` 发送数据

### 注意事项

- 确保防火墙允许程序访问网络
- 目标 IP 必须与 Hi3861 在同一局域网
- 某些杀毒软件可能误报，请添加信任
