# OpenHarmony Hi3861 开发环境搭建指南

本指南帮助你在 Windows 环境下搭建 Hi3861 开发编译环境。

## 方案一: WSL2 + Ubuntu (推荐)

### 准备工作

1. **启用 WSL2**
   ```powershell
   # 以管理员身份运行 PowerShell
   wsl --install
   ```

2. **安装 Ubuntu 20.04 或 22.04**
   ```powershell
   wsl --install -d Ubuntu-22.04
   ```

3. **克隆 OpenHarmony SDK**
   ```bash
   git clone https://gitee.com/hihope_iot/hi3861_lab.git ~/hi3861_lab
   cd ~/hi3861_lab
   ```

### 安装编译工具

```bash
sudo apt update
sudo apt install -y build-essential gcc g++ make git python3 python3-pip
pip3 install scons
pip3 install --upgrade pip
pip3 install pycryptodome ecdsa
```

### 编译

```bash
cd ~/hi3861_lab
hb build -f
```

## 方案二: 虚拟机 Ubuntu

1. 安装 VirtualBox 或 VMware Workstation
2. 安装 Ubuntu 20.04/22.04
3. 在虚拟机中配置共享文件夹或使用 SFTP 传输文件
4. 同上安装编译工具和执行编译

## 烧录

### 使用 HiBurn (Windows)

1. 下载 HiBurn: https://www.hihope.org/download/HiBurn.aspx
2. 连接开发板 (USB 转串口)
3. 打开 HiBurn，选择正确的串口 (COMxx)
4. 选择编译生成的 `.bin` 文件
5. 设置波特率 115200
6. 点击 "Connect" 连接
7. 点击 "Burn" 开始烧录
8. 烧录完成后点击 "Disconnect"

### 串口工具

推荐使用 SSCOM 或 Mobaxterm，波特率 115200，8N1。

## 代码开发流程

```
1. 在 Ubuntu (WSL/VM) 中编辑代码
2. 执行 hb build -f 编译
3. 将 out/ 目录下的 .bin 文件复制到 Windows
4. 用 HiBurn 烧录
5. 用串口工具观察输出
```

## WSL2 串口访问

如果需要通过 WSL 直接烧录或访问串口:

```powershell
# 安装 USBIPD-WIN
winget install --accept-package-agreements --accept-source-agreements usbipd-win

# 列出 USB 设备
usbipd list

# 绑定设备到 WSL
usbipd bind --busid <BUSID>

# 在 WSL 中挂载
usbipd attach --busid <BUSID> --wsl
```

## 常见问题

### Q: hb 命令找不到

```bash
pip3 install -g ohos-build
# 或
source build/env_setup.sh
```

### Q: 编译报错缺少头文件

确保所有依赖库已正确下载:
```bash
git submodule update --init --recursive
```

### Q: HiBurn 识别不到设备

1. 检查 USB 线是否为数据线（不是充电线）
2. 检查是否安装了 USB 转串口驱动 (CH340/FTDI)
3. 尝试重新插拔 USB
4. 点击 Connect 后再按复位键

### Q: 串口无输出

1. 检查波特率是否为 115200
2. 检查串口工具是否正确选择 COM 口
3. 检查开发板是否正常启动（LED 指示灯）

## 华为云 IoTDA 配置

### 注册华为云账号

1. 访问 https://www.huaweicloud.com/
2. 注册并实名认证
3. 进入 IoTDA 服务

### 创建 IoTDA 实例

1. 进入 "设备接入" 服务
2. 创建 "标准版" 实例
3. 记下 **接入地址**

### 注册设备

1. 进入设备管理
2. 点击 "注册设备"
3. 填写设备信息并保存 **设备密钥**

### 生成 MQTT 密码

使用 Python 脚本生成:

```python
import hmac
import hashlib
from datetime import datetime

device_id = "你的设备ID"
device_secret = "你的设备密钥"
client_id = f"{device_id}_0_0_{int(datetime.now().timestamp())}"

# 签名内容
auth_content = f"deviceId{device_id}timestamp{int(datetime.now().timestamp())}clientId{client_id}versionv3.0"
password = hmac.new(device_secret.encode(), auth_content.encode(), hashlib.sha256).hexdigest()
print(f"ClientID: {client_id}")
print(f"Username: {device_id}")
print(f"Password: {password}")
```
